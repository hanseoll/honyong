# 한설 방송용 페이지 4단계 운영 가이드

## 이번 단계에서 추가된 기능
- 상단 `새로고침` 버튼
- 구글시트 바로가기 `관리 시트` 버튼
- 마지막 동기화 시각 표시
- 5분 자동 새로고침(설정 가능)
- 인터넷이 잠깐 끊겨도 기본 화면을 여는 오프라인 캐시
- 아이폰·아이패드 홈 화면 추가용 PWA 유지

## config.js에서 꼭 입력할 값
```js
spreadsheetId: "구글시트_ID",
adminSheetUrl: "구글시트_전체주소",
autoRefreshMinutes: 5
```

## 운영할 때
1. 구글시트 내용을 수정합니다.
2. 페이지에서 `새로고침`을 누릅니다.
3. 또는 5분이 지나면 자동으로 다시 불러옵니다.

## 아이패드 홈 화면에 추가
Safari에서 배포된 페이지 열기 → 공유 버튼 → 홈 화면에 추가
