# 11단계 관리자 페이지 설치

## 포함된 기능
- 업보 추가
- 업보 수행 차감
- 업보 남은 수량 직접 변경
- 같은 닉네임·같은 업보 중복 행 자동 합치기
- 수량이 0 이하이면 해당 업보 삭제
- 현재 업보 목록 확인
- 호뇽단 등록·수정
- 공지 추가
- 스케줄 추가
- 게임 추가·수정

공개 홈페이지에는 관리자 링크가 표시되지 않습니다.

---

## 1. GitHub에 11단계 파일 올리기

ZIP 압축을 풀고 안쪽 파일들을 GitHub 저장소에 덮어 올린 뒤
`Commit changes`를 누릅니다.

배포 후 관리자 주소:

`https://honyong-broadcast.pages.dev/admin.html`

---

## 2. Apps Script 설치

1. 관리 중인 구글 스프레드시트를 엽니다.
2. `확장 프로그램 → Apps Script`를 누릅니다.
3. 기존 Code.gs 내용을 지웁니다.
4. ZIP 안의 `관리자_API_AppsScript.gs` 내용을 전부 붙여넣습니다.
5. 코드 위쪽의 `setAdminSecret()` 함수에서
   `여기를_긴_관리자_비밀번호로_바꾸세요` 부분을
   본인만 아는 긴 비밀번호로 바꿉니다.
6. 상단 함수 선택에서 `setAdminSecret`을 선택하고 실행합니다.
7. 처음 실행할 때 Google 권한을 허용합니다.

---

## 3. 웹 앱으로 배포

1. Apps Script 오른쪽 위 `배포 → 새 배포`
2. 유형 선택에서 `웹 앱`
3. 실행 사용자: `나`
4. 액세스 권한: `모든 사용자`
5. `배포`
6. 생성된 주소 중 `/exec`로 끝나는 주소를 복사합니다.

코드를 나중에 수정했다면 기존 배포의 `새 버전`으로 다시 배포해야 합니다.

---

## 4. 관리자 페이지 연결

1. `https://honyong-broadcast.pages.dev/admin.html` 접속
2. Apps Script 웹 앱 주소 입력
3. setAdminSecret에서 지정한 관리자 비밀번호 입력
4. `설정 저장·연결 확인`
5. 연결됨 표시가 뜨면 사용 시작

공용 컴퓨터에서는 `현재 창에서만 사용`을 선택하세요.

---

## 보안 주의

- 관리자 페이지 주소와 비밀번호를 방송 화면이나 채팅에 노출하지 마세요.
- 비밀번호는 사이트 코드에 저장되지 않고 Apps Script 서버에서 검사합니다.
- Apps Script 웹 앱은 `모든 사용자` 접근으로 배포하지만,
  올바른 관리자 비밀번호가 없으면 데이터를 수정하지 못합니다.
