호뇽 방송 관리 시스템 설정

v1.0 최종 설치 점검용 페이지입니다. 아래 순서대로 한 번만 확인하세요.

1구글시트 ID 확인

예: docs.google.com/spreadsheets/d/여기/edit

2시트 공개 확인

구글시트에서 파일 → 공유 → 웹에 게시를 완료해야 공개 페이지가 데이터를 읽을 수 있습니다.

ID를 입력한 뒤 테스트하세요.
3관리자 Apps Script 확인
웹 앱 주소와 비밀번호를 입력하세요.
4최종 확인

설정값은 이 브라우저의 관리자 페이지용 저장소에만 저장됩니다. 공개 사이트의 config.js는 마지막 배포 때 직접 반영해야 합니다.
